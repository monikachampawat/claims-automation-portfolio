
# User Stories & Acceptance Criteria

**US101** – As a claims agent, I want to see pending claims with SLA status so I can prioritize work.
- **AC:** Grid shows ID, status, priority, SLA due, owner.

**US102** – As a supervisor, I want KPIs (Volume, Avg Time, SLA %, Aging) so I can monitor performance.
- **AC:** Dashboard refreshes daily; supports date filters.

**US103** – As compliance, I want an audit trail of all status changes.
- **AC:** Claim_Event records EventType and timestamp per change.
